## 📂 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/subhashree0454/Portfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Build for Production**:
   ```bash
   npm run build
   ```
